  2. 本教程一共分为 5 个部分

    1. Part A： 导入代码库，这是老师已经写好的代码。
    2. Part B： 制作容器，创建（new）和输入变量。
    3. Part C： 调用容器的方法。
    4. Part D： 老板制造方法 generateFunction 和 loopFunction。
    5. Part E： 开始游戏方法和动画（每一帧）的方法。

  3. Part A

    这里是老师已经写好的代码，学生们只需调用即可。
    只需记得调用的名字（比如 MyGame），这编辑软件会自动“提示”你，接着只需选择提示就完成了。

    class Coin extends Enemy 比较复杂，解释：制作一个新的类（class），扩展 Enemy 的类，取名叫“Coin”

    *注意大小写

  4. Part B

    const 是容器，给容器取个名字 （const myGame）。
    =（等于）是把右边的东西转进左边。
    new 是创建，紧接着的是背起来的“类”。
    ()（香蕉）是放入“变量”。

    变量是指改变结果的内容，比如改变坐标，宽度高度等。

  5. Part C

    调用容器，比如 myGame.setTitle('跳跳吃金币')
    翻译成中文比较好理解 “我的游戏 的 设置主题 成 跳跳吃金币”
